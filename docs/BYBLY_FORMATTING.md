# BYBLY FORMATTING GUIDE

## LOGS - ASCII EMOTICONS ONLY

For logs use only ASCII emoticons:

```
[i] - info
[-] - error  
[+] - success
[!] - warning
[*] - debug
[?] - question
[#] - hash/processing
[@] - mention/notification
[~] - wave/connection
[^] - up/positive
[v] - down/negative
[>] - forward/next
[<] - backward/previous
[=] - equal/stable
[%] - percentage/progress
[&] - and/connection
[|] - pipe/flow
[\] - backslash/escape
[/] - slash/division
[.] - dot/point
[,] - comma/separator
[;] - semicolon/end
[:] - colon/start
[)] - happy
[(] - sad
[0] - zero/null
[1] - one/single
[2] - two/double
[3] - three/triple
[4] - four/quad
[5] - five/penta
[6] - six/hexa
[7] - seven/hepta
[8] - eight/octa
[9] - nine/nona
```

## MESSAGES - NON-STANDARD EMOJIS
